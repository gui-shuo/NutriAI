## 问题排查

如果测试失败，请检查：

### 问题1：按钮点击无反应
**可能原因**：
- 遮罩层仍然存在
- CSS样式阻止了点击
- JavaScript事件被阻止

**排查步骤**：
1. 打开开发者工具 → Elements
2. 检查是否有`<div class="el-overlay">`元素
3. 检查该元素的CSS样式
4. 检查是否有`pointer-events: none`

### 问题2：弹窗无法显示
**可能原因**：
- Dialog/MessageBox被CSS隐藏
- z-index层级问题

**排查步骤**：
1. 打开开发者工具 → Elements
2. 搜索`.el-dialog__wrapper`或`.el-message-box__wrapper`
3. 检查元素的`display`和`visibility`属性
4. 检查`z-index`值

### 问题3：页面无法滑动
**可能原因**：
- 遮罩层覆盖了整个页面
- body元素被设置了`overflow: hidden`

**排查步骤**：
1. 打开开发者工具 → Elements
2. 检查`<body>`元素的样式
3. 检查是否有全屏的遮罩层
4. 检查`overflow`属性
