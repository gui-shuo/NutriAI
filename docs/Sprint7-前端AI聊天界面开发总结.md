## 🔧 技术实现细节

### 1. 消息数据结构
```javascript
{
  id: 'msg_1234567890_abc',      // 唯一ID
  role: 'user' | 'assistant',    // 角色
  content: '消息内容',            // 内容
  timestamp: 1701234567890,      // 时间戳
  loading: false,                // 加载状态
  file: {                        // 文件信息（可选）
    name: 'file.txt',
    size: 1024,
    type: 'text/plain'
  }
}
```

### 2. Markdown渲染流程
```
用户输入 → marked解析 → DOMPurify清理 → v-html渲染
```

### 3. 自动滚动逻辑
- 新消息添加时自动滚动
- 用户主动滚动时不打断
- 距离底部>200px时显示滚动按钮
- 点击按钮平滑滚动到底部

### 4. 键盘事件处理
- Enter: 发送消息
- Shift + Enter: 换行
- Ctrl/Cmd + K: 清空对话
- Ctrl/Cmd + E: 导出对话
