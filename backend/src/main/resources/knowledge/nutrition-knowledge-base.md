# NutriAI 营养知识库

## 知识库数据来源说明

本知识库数据来源于以下中国国内公开营养数据源：

### 一、数据获取方式

1. **中国食物成分表（第6版/标准版）**
   - 来源：中国疾病预防控制中心营养与健康所 编著
   - 获取：可通过各大学术数据库或国家标准网站查询
   - 内容：覆盖2000+种中国常见食材的详细营养成分（能量、蛋白质、脂肪、碳水化合物、维生素、矿物质等）
   - 标准号：GB/T 编号体系

2. **中国居民膳食指南（2022版）**
   - 来源：中国营养学会
   - 获取：http://dg.cnsoc.org/ （中国营养学会官网）
   - 内容：中国居民膳食营养素参考摄入量（DRIs）、膳食宝塔

3. **国家食品安全风险评估中心数据**
   - 来源：国家食品安全风险评估中心
   - 获取：https://www.cfsa.net.cn/
   - 内容：食品营养成分检测数据

4. **中国知网（CNKI）营养学文献**
   - 获取：https://www.cnki.net/
   - 搜索关键词：食物营养成分、膳食营养、中国食物成分
   - 内容：权威学术论文中的营养数据

5. **FoodDB / USDA食品数据库（补充参考）**
   - FoodDB: https://foodb.ca/
   - USDA: https://fdc.nal.usda.gov/
   - 内容：全球食品营养成分数据，可作为中国数据的补充参考

### 二、配置方式

本项目通过以下方式使用营养知识库：
1. **数据库内置数据**：food_nutrition表中存储了常见中国食物的营养成分数据
2. **AI提示词注入**：将核心营养知识作为系统提示词注入AI对话上下文
3. **实时查询**：AI对话中可引用知识库数据进行个性化推荐

### 特殊人群营养建议

#### 减脂人群
- 热量缺口：每日减少300-500kcal
- 蛋白质：1.2-1.6g/kg体重
- 碳水化合物：3-5g/kg体重
- 脂肪：0.5-1g/kg体重
- 建议每周减重0.5-1kg

#### 增肌人群
- 热量盈余：每日增加300-500kcal
- 蛋白质：1.6-2.2g/kg体重
- 碳水化合物：4-7g/kg体重
- 脂肪：0.5-1.5g/kg体重

#### 孕期营养
- 孕早期：+0kcal/天
- 孕中期：+300kcal/天
- 孕晚期：+450kcal/天
- 叶酸：600μg/天
- 铁：20-24mg/天
- 钙：1000mg/天

#### 糖尿病饮食
- 碳水化合物：占总能量45-60%
- 膳食纤维：25-30g/天
- 低GI食物优先
- 少食多餐，定时定量

### 食物GI值参考
| 分类 | GI值范围 | 代表食物 |
|------|----------|----------|
| 低GI | ≤55 | 燕麦、荞麦、大部分蔬菜、豆类 |
| 中GI | 56-69 | 全麦面包、糙米、红薯 |
| 高GI | ≥70 | 白米饭、白面包、土豆 |

### 中国人常见营养缺乏
1. **钙**：推荐每日800mg，多数人摄入不足400mg
2. **维生素D**：日照不足地区普遍缺乏
3. **铁**：育龄女性缺铁性贫血发病率约20%
4. **膳食纤维**：推荐25-30g/天，平均摄入仅11g
5. **维生素B族**：精加工米面导致B1、B2流失

### 饮食误区纠正
1. "少吃多餐减肥"→ 关键是总热量控制
2. "不吃主食减肥"→ 会导致酮症酸中毒，碳水不可少于130g/天
3. "果汁等于水果"→ 果汁丢失膳食纤维，升糖更快
4. "喝骨头汤补钙"→ 骨头汤含钙极少，含油脂多
5. "蛋黄胆固醇高不能吃"→ 正常人每天1个全蛋无害
